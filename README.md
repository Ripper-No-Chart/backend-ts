# Typescript Backend
## _Basic typescript backend_

[![N|Solid](https://cldup.com/dTxpPi9lDf.thumb.png)](https://nodesource.com/products/nsolid)

A basic backend, with user crud, validation by token and hashing by bcrypt

- Typescript
- JWT
- Bcrypt
- ts-node
- ExpressJS
- Mongoose
- Moment
- Chalk

## Usage

- npm install
- (dev) npm run dev (3001)
- (build) npm run build
